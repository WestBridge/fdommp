项目起源
==
根据日常工作及以往项目经验,通过学习和借鉴写出的运维平台,已在50+的机器里面使用与测试,其中有些代码很笨拙,但随着经验提升相信会更好!

技术序列
==
### 前端
  * Bootstrap/JQuery/VUE
### 后台
  * Django/python3.6+

运行环境
==
### 容器运行
  * Docker version 19.03.2 ++
  * docker-compose version 1.8.0 ++
  
功能清单
==
  ### 资产管理
      * 服务端拉取数据（手自一体），客户部部署脚本推送
  ### 主机管理
      * 运行情况
      * 网页SSH，SHELL,PLAYBOOK,动态CRONTAB
  ### 服务管理
      * 服务的状态
      * 服务的启停
